# 图书管理系统 java1179

## 项目概述

图书管理系统是一个基于SpringBoot和Vue的前后端分离项目，旨在为图书馆的日常运营提供一个高效、便捷的信息化解决方案。

## 技术栈

- 前端：Vue
- 后端：SpringBoot
- 开发环境：IDEA、JDK 1.8、Maven、MySQL 8.0、Node.js 16

## 功能模块

### 系统角色

- 普通用户
- 管理员

### 主要功能

- **图书信息管理**：对图书的基础信息进行增、删、改、查操作。
- **图书类别管理**：管理图书的分类，包括分类的增、删、改、查。
- **借阅信息管理**：记录和管理用户的借阅信息。
- **用户管理**：对系统用户进行管理，包括用户信息的增、删、改、查。
- **修改密码**：允许用户自主修改账户密码。
- **用户借书**：用户可以通过系统借阅图书。
- **用户还书**：用户通过系统归还图书，更新借阅信息。

## 运行环境

- **后端**：Java 1.8及以上版本，Maven构建工具，MySQL数据库
- **前端**：现代浏览器，Node.js环境

## 部署步骤

1. 准备后端运行环境，包括Java、Maven、MySQL。
2. 使用Maven编译后端项目。
3. 准备前端运行环境，安装Node.js，并编译前端项目。
4. 部署前端代码到Web服务器。
5. 部署后端代码到应用服务器，并配置数据库连接。
6. 确保前端与后端网络通信正常。

## 核心亮点

- **前后端分离**：后端提供RESTful API，前端负责界面展示，分工明确，易于维护。
- **高效开发**：基于SpringBoot和Vue的现代化开发框架，提高开发效率。
- **灵活部署**：支持跨平台部署，易于拓展和升级。
